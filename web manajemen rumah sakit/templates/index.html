Modul Operasional Struktur Data / Algoritma Time Complexity (Worst Case) Keterangan Implementasi Teoretis
Pendaftaran Antrean Queue (FIFO List) O(1) Menambahkan data pasien baru ke ujung belakang barisan (*Rear*).
Gudang Resep Apotek Stack (LIFO List) O(1) Menaruh dan mengambil resep dari tumpukan paling atas (*Top*).
